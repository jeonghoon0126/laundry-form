## 동선 특이사항

- `왕산로 200, 1004호`는 2026-04-09(목) 1회 추가, 2026-04-13(월)부터 2주 간격 월요일마다 포함

## 배포

push → GitHub Actions 자동 실행
워크플로우: .github/workflows/send-route-sms.yml

## 테스트

TEST_DATE=YYYY-MM-DD → 특정 날짜 데이터로 테스트
